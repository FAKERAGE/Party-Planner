# Event & Party Planner -- V1 PRD

## Project Overview
I want to generate a quick, functional Event & Party Planner web app:
- Types of events: birthday parties, family gatherings, hangouts.
- Primary user: mainly for everyone but if I had to pick than kids/teens.

## Core Features
- [ ] Event creation form (name, date, location, description)
- [ ] Theme selector with at least 4 party themes
- [ ] Guest list manager with add/remove and RSVP tracking
- [ ] Activity planner with drag-and-drop or similar interaction (allow custom activities)
- [ ] Event summary view showing all details together
- [ ] All code in a single HTML file with embedded CSS and JS

## What To Avoid
- Do NOT add user accounts, login, databases, or backend
- Do NOT add payment processing or ticket sales
- Do NOT add email/SMS notifications
- Keep it simple -- this is a FIRST DRAFT, not a finished product

## Design Notes
- Style: minimal design
- Color preferences/themes: not specified, keep clean and minimal
